# Local Graph Builder

# Env

```sh
pip install requirements.txt

# 可能遇到 torch-mlir 的报错  手动指定一下
pip install --pre torch-mlir torchvision -f https://github.com/llvm/torch-mlir-release/releases/expanded_assets/dev-wheels
```
